--- METHODOLOGY ---
1. For each global address, queried idautils.XrefsTo() via IDA Python
2. Sampled up to 25 xrefs per global (distributed across the xref list)
3. For each sampled xref, decoded the instruction and next 5 instructions
4. Classified each xref site using:
   - int_cmp: next instruction is cmp/test/arithmetic
   - bool_flag: int_cmp + conditional jump
   - float_op: next instruction is fld/fstp/fcomp/etc
   - ptr_deref: next instruction dereferences via [reg+offset]
   - func_ptr: loaded value used for indirect call
   - array_base: LEA instruction
   - string_ptr: push of global address
5. Aggregated classification counts and determined observed_type via priority rules
